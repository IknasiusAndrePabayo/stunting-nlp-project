# Sistem Prediksi Stunting Balita ðŸ‘¶

Ini adalah demo aplikasi Machine Learning untuk mendeteksi dini stunting pada balita menggunakan algoritma **Random Forest** dan **PySpark**.

## Cara Penggunaan
1. Masukkan **Umur** balita (bulan).
2. Masukkan **Tinggi Badan** (cm).
3. Pilih **Jenis Kelamin**.
4. Klik tombol **Submit** untuk melihat hasil prediksi.

## Teknologi
- **Model:** Random Forest Classifier
- **Engine:** Apache Spark (PySpark)
- **Interface:** Gradio
